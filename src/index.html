QQuizzes API Document

About

Welcome to the QQuizzes API document! This API serves as the backend for the Quizzes web application, providing functionalities for managing quizzes, users, authentication, and more.

End Point

http://api-qquizzes-b24b27944f64.herokuapp.com/api

API Call

Authentication

Register

POST http://api-qquizzes-b24b27944f64.herokuapp.com/api/auth/register
- Request Body:
  • username: String (required) - Username of the user.
  • password: String (required) - Password of the user.
  • email: String (required) - Email of the user.

Login

POST http://api-qquizzes-b24b27944f64.herokuapp.com/api/auth/login
- Request Body:
  • username: String (required) - Username of the user.
  • password: String (required) - Password of the user.

Refresh Token

GET http://api-qquizzes-b24b27944f64.herokuapp.com/api/auth/refreshToken

Logout

GET http://api-qquizzes-b24b27944f64.herokuapp.com/api/auth/logout

Quiz

Get owned quizzes

GET http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/admin

Get public quizzes

GET http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/public

Get quizzes participated

GET http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/admin/quizzesParticipated

Get quiz by id

GET http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/{id}

Create a quiz

POST http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes
- Request Body:
  • quizName: String (required) - Name of the quiz.
  • topic: String (required) - Topic of the quiz.
  • Other optional fields specific to your application.

Update a quiz

PUT http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/{id}
- Request Body:
  • Same as create quiz endpoint.

Delete a quiz

DELETE http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/{id}

Question

Get all questions from a quiz

GET http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/{quizId}/questions

Get all questions from a quiz which created by user

GET http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/{quizId}/questions/private

Create a question

POST http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/{quizId}/questions
- Request Body:
  • question: String (required) - The question text.
  • selections: Array (required) - Array of selections with at least 4 items.
  • Other optional fields specific to your application.

Update a question

PUT http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/{quizId}/questions/{id}
- Request Body:
  • Same as create question endpoint.

Delete a quiz

DELETE http://api-qquizzes-b24b27944f64.herokuapp.com/api/quizzes/{quizId}/questions/{id}
